# GitHub Profile Achievements 🏆

## Descrição

Este repositório é uma coleção que lista todas as conquistas disponíveis no perfil do GitHub. Esse repositório só existia em inglês, porém dicidir traduzir todo o conteúdo para o português.

Foi criado após o lançamento do primeiro Helicóptero em Marte, o [Ingenuity](https://en.wikipedia.org/wiki/Ingenuity_(helicopter)). O GitHub anunciou a nova seção de Conquistas, que inclui a medalha da Missão do Helicóptero Mars 2020, a medalha do Arctic Code Vault e uma medalha para patrocinar trabalhos de código aberto via GitHub Sponsors.

Se você tiver mais informações sobre conquistas não listadas aqui ou encontrar algo incorreto, sinta-se à vontade para abrir uma issue.

Se você não gosta das Conquistas e deseja desativá-las, pode seguir [este link](https://docs.github.com/account-and-profile/setting-up-and-managing-your-github-profile/managing-contribution-settings-on-your-profile/showing-your-private-contributions-and-achievements-on-your-profile#changing-the-visibility-of-achievements) para configurá-las.
